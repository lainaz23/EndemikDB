# EndemikDB
Aplikasi Android untuk menampilkan data hewan dan tumbuhan endemik Indonesia.

## Dibuat oleh
**Laila Nazmi**

## Fitur
- Splash Screen
- Home (Hewan & Tumbuhan)
- Filter Region
- Detail Spesies
- Favorit
- Pencarian
- Dark & Light Mode
